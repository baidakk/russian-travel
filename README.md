# Проект: Путешествие по России

### Описание
Одностраничный сайт про путешествия по России. С помощью этого сайта вы увидите путь путешествия и самые красивые места по пути следования:

* Куршская коса
* Кольский полуостров
* Алтай
* Байкал
* Карелия


**Функциональность**

В прокте использованы технологии :

* Верстка на Flex
* Вертска на Grid
* Адаптивная вертска
* Структура по БЭМ
